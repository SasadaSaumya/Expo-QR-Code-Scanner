## Installation

1. Clone the repository

   - ```
     git clone https://github.com/SasadaSaumya/Expo-QR-Code-Scanner.git
     cd Expo-QR-Code-Scanner
     ```

2. Install dependencies

   - ```
     npm install
     ```

3. Start the app

   - ```
     npx expo start
     ```
